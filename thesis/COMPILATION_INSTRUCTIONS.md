# LaTeX Compilation Instructions

## Important: Bibliography Compilation

Since we're using **biblatex** (not BibTeX), you MUST use **biber** instead of **bibtex** to process the bibliography.

## Correct Compilation Sequence

### Option 1: Manual Compilation (Recommended)

```bash
cd thesis

# Step 1: First pdflatex pass
pdflatex bachelor.tex

# Step 2: Run biber (NOT bibtex!)
biber bachelor

# Step 3: Second pdflatex pass
pdflatex bachelor.tex

# Step 4: Third pdflatex pass (for cross-references)
pdflatex bachelor.tex
```

### Option 2: Using Your LaTeX Editor

**In TeXstudio/TeXworks/Overleaf:**

1. Set bibliography tool to **biber** (not bibtex)
2. Compile sequence: **pdflatex → biber → pdflatex → pdflatex**

**In Overleaf:**
- Go to Menu → Compiler → **pdfLaTeX**
- Go to Menu → Bibliography → **biber** (not BibTeX)
- Compile multiple times

**In TeXstudio:**
- Options → Configure TeXstudio → Build → Default Bibliography → Change to **biber**

### Option 3: Using latexmk (Automatic)

```bash
latexmk -pdf -pvc -bibtex- -biber bachelor.tex
```

The `-bibtex-` flag disables bibtex, and `-biber` enables biber.

## Troubleshooting

### Issue: Bibliography not showing

**Problem**: Using bibtex instead of biber

**Solution**: 
1. ✅ Delete `.bbl` and `.blg` files
2. ✅ Run `biber bachelor` (not `bibtex bachelor`)
3. ✅ Recompile with pdflatex

### Issue: Citations show as [?]

**Problem**: Bibliography not processed

**Solution**:
1. ✅ Run biber: `biber bachelor`
2. ✅ Recompile: `pdflatex bachelor.tex` (twice)

### Issue: "biber: command not found"

**Problem**: biber not installed

**Solution**:
- **Windows (MiKTeX)**: Should be included, or install via MiKTeX Package Manager
- **Windows (TeX Live)**: Should be included
- **macOS**: `brew install biber` or included with MacTeX
- **Linux**: `sudo apt-get install biber` or included with TeX Live

### Issue: DOIs not showing

**Problem**: biblatex not configured correctly

**Solution**:
1. ✅ Verify `\usepackage[doi=true]{biblatex}` in preamble
2. ✅ Check `references.bib` has `doi = {...}` fields
3. ✅ Recompile with biber

## Quick Check

After compilation, verify:
- [ ] Bibliography section appears at end
- [ ] Citations are numbered (e.g., [1], [2])
- [ ] DOIs are visible in bibliography entries
- [ ] No `[?]` citations in text

## Files Generated

After successful compilation:
- `bachelor.bbl` - Bibliography file (generated by biber)
- `bachelor.bcf` - Biber control file
- `bachelor.blg` - Biber log file
